Web Design in 4 Minutes

1. Create a Basic HTML Skeleton

Start with this simple structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

2. Add Some Style

Use CSS to make the site look clean and modern:

body {
  font-family: Arial, sans-serif;
  color: #333;
  background: #f4f4f4;
}
h1 {
  color: #3498db;
}

3. Improve Spacing and Layout

Use div and padding for better spacing:

.container {
  width: 80%;
  margin: auto;
  padding: 20px;
  background: #fff;
  box-shadow: 0 0 10px #ccc;
}

4. Use Links and Buttons

Make navigation and interactions easy with buttons:

<a href="https://example.com" class="btn">Visit Example</a>
Visit Example

5. Add Images

Include images to make the page visually appealing:

<img src="https://via.placeholder.com/300" alt="Sample Image" />

6. Create Navigation

Use nav for easy navigation:

<nav>
  <a href="#section1">Section 1</a> |
  <a href="#section2">Section 2</a>
</nav>

7. Make It Responsive

Use media queries to adjust for different screen sizes:

@media (max-width: 768px) {
  .container {
    width: 100%;
  }
}
